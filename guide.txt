git init -> Создать в этой папке ГИТ репозиторий
git status -> Проверить статус ГИТа(что происходит)
git add <ИМЯ ФАЙЛА/ПАПКИ> -> Добавляет в коммит файл/папку(Чтобы добавить все, нужно писать git add .)
git commit -m '<СООБЩЕНИЕ>' ->
git push -> Отправить все изменения на сервер ГИТа
git remote add origin <URL> -> Связать папку с ГИТ репозиторием по <URL>
git push --set-upstream origin master -> Настроить push чтобы заливалось в главную ветку
